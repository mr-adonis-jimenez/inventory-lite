# Security Policy

## Supported Versions
The `main` branch is actively maintained.

## Reporting a Vulnerability
Please open a **private** advisory on GitHub Security Advisories or email security@example.com. Include:
- Affected commit/branch and steps to reproduce.
- Impact assessment and potential fixes.

## Branch Protection
- Merges to `main` require PRs, 2 approvals, passing checks (`build`, `test`, `lint`), code-owner review, and resolved conversations.
- Force-pushes and deletions disabled. Signed commits required.
